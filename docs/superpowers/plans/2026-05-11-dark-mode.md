# Dark Mode Support Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Adapt the YT History Cleaner panel to render correctly in YouTube's dark mode by appending a `html[dark]` CSS override block to the existing `STYLES` string.

**Architecture:** A single block of CSS selectors prefixed with `html[dark] #ytc-panel` is appended to the `STYLES` constant. YouTube sets the `dark` attribute on `<html>` synchronously when the user toggles dark mode, so CSS reacts instantly — no JS changes required.

**Tech Stack:** Vanilla JS userscript (Tampermonkey/Violentmonkey), CSS attribute selectors.

---

### Task 1: Append dark mode CSS overrides to `STYLES`

**Files:**
- Modify: `yt-history-cleaner.user.js` — append CSS block inside the `STYLES` template literal (after the closing `@keyframes ytc-fadein` block, before the closing backtick)

The spec covers 10 component groups. All changes are additive — no existing CSS is touched.

- [ ] **Step 3: Verify in YouTube dark mode — Quick mode**

1. In Tampermonkey/Violentmonkey, update the script with the saved file.
2. Go to `youtube.com/feed/history`.
3. Open YouTube Settings → Appearance → Dark theme. Enable it.
4. The panel should show:
   - Background `#212121`, border `#3d3d3d`
   - Title text white (`#f1f1f1`), red play icon unchanged
   - "Quick" active button: dark blue `#1557b0`, white text
   - "Custom Date" inactive button: `#2d2d2d` bg, `#aaa` text
   - Dropdown: `#2d2d2d` bg, white text
   - "Scan" button: unchanged blue `#1a73e8`

- [ ] **Step 4: Verify in YouTube dark mode — Custom Date / Calendar mode**

1. Click "Custom Date".
2. The calendar should show:
   - Calendar area: `#1a1a1a` bg, `#3d3d3d` border
   - Month label: `#f1f1f1`
   - Day-of-week headers: `#666`
   - Calendar cells: `#e8e8e8` text; hover shows `#333` bg
   - Spillover and future dates: `#555` (dimmed)
   - Today: `#8ab4f8` bold text with `#8ab4f8` dot
3. Click a start date and an end date. Verify:
   - Range endpoints: `#1a73e8` bg, white text
   - In-range cells: `#1a3a6e` bg, `#8ab4f8` text
   - Summary box: `#1a3a6e` bg, `#8ab4f8` text
4. Click Scan. Verify the scanning info box is `#1a3a6e`/`#8ab4f8`.

- [ ] **Step 5: Verify light mode is unaffected**

1. Switch YouTube back to Light theme.
2. Reload `youtube.com/feed/history`.
3. The panel should look exactly as before — white background, `#e0e0e0` borders, original blue/grey text. None of the light mode styles should have changed.
